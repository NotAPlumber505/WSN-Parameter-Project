# Wireless Sensor-Actuator Networks (WSANs) Visualization Hub

Welcome to the **Wireless Sensor-Actuator Networks (WSANs) Visualization Hub**! This platform is designed to revolutionize the configuration and adaptation of industrial Wireless Sensor-Actuator Networks (WSANs). Built upon a decade of research and real-world deployments of wireless standards like **WirelessHART** and **ISA100**, this project provides a powerful toolset to visualize, analyze, and optimize complex industrial networks.

Our goal is to help users better understand, configure, and adapt their wireless networks in real-time by utilizing cutting-edge machine learning techniques, deep learning, and wireless simulations. Whether you're a researcher, industry professional, or just exploring the future of wireless networks, this platform provides an interactive experience with real-time insights.

## Features

- **Real-time Visualization**: View the performance of industrial WSANs as they adapt to changing conditions.
- **Data Exploration**: Interactive charts, graphs, and tables to help you explore insights from both simulations and real-world testbeds.
- **Advanced Network Configuration**: Optimize network configurations using machine learning-powered analysis.
- **Seamless Integration**: Compare data across multiple sources with easy-to-use charts and tables.

## Installation and Setup

### Prerequisites

Ensure you have the following installed:
- **Python 3.8+**
- **pip** (Python package manager)

### 1. Clone the Repository

```
git clone https://github.com/yourusername/WSAN-Visualization-Hub.git
cd WSAN-Visualization-Hub
```

### 2. Set Up a Virtual Environment (Optional but Recommended)

```
python -m venv venv
```
Activate the virtual environment:
Windows
```
venv\Scripts\activate
```
macOS/Linux
```
source venv/bin/activate
```

### 3. Install Required Dependencies
```
pip install -r requirements.txt
```

### 4. Start the Application
```
streamlit run app.py
```

A local URL will be displayed, for example:
```
http://localhost:8501
```

## Usage

### 1. Explore Data and Visualize Networks

After running the application, you will be presented with several interactive features:

- **Real-time Visualizations:** Monitor the performance of your WSANs.
- **Data Exploration:** Dive into network data through charts and graphs to understand the network's behavior and performance.
- **Advanced Configuration:** Use machine learning algorithms to adjust and optimize the configuration of your wireless networks.

### 2. Analyze Network Performance

- Navigate through the **Publications** tab to explore advanced research papers and their related results. Each publication is linked to the relevant data and source code for deeper exploration.
- Use the **Simulation and Configuration Tools** to compare how the system adapts to different conditions based on real-time data input.

### 3. Modify or Extend the Application

If you'd like to modify or extend the application, the code is open-source! Feel free to make changes or add new features, such as:
- Enhancing the data visualization
- Adding new machine learning models
- Integrating new data sources
